# Subagent: Vertical Volumes

## Зона ответственности
Анализ вертикальных объемов по USDC futures-инструментам в live-режиме.

## Входы
- поток рыночных данных;
- ограничения из `workspace_start/ТЗ.txt`;
- контракт интеграции из `specs/subagent_contracts.md`.

## Выход
- оценка сигналов по вертикальным объемам;
- обоснование веса сигнала;
- артефакт в `artifacts/` с временной меткой.
