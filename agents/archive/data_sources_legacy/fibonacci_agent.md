# Subagent: Fibonacci Agent

## Допустимые входные данные
- локальные swing high / swing low по инструменту;
- подтверждённые экстремумы выбранного таймфрейма (`5m`, `15m`, `1h`, `4h`);
- текущая цена и направление базового импульса;
- временная метка расчёта.

## Что считает субагент
Субагент строит уровни коррекции и расширения Фибоначчи по подтверждённому импульсу и оценивает вероятность реакции цены на ключевых уровнях.

## Возвращаемые признаки и метрики
- `fib_anchor_high`
- `fib_anchor_low`
- `nearest_retracement_level`
- `nearest_extension_level`
- `level_reaction_strength`
- `direction_candidate`
- `recommended_grid_bounds`
- `suggested_tp`
- `suggested_sl`
- `invalidators`

## Формат confidence score
```json
{
  "score": 0.0,
  "scale": "0.0-1.0",
  "label": "low|medium|high"
}
```
`score` зависит от качества swing-структуры и силы реакции на ближайшие уровни.

## Запрещённые данные для принятия решения
- неподтверждённые экстремумы из произвольной разметки;
- волновая интерпретация вне собственных уровней Фибоначчи;
- новостной и фундаментальный фон;
- сигналы нерелевантных модулей.

## Допустимая частота обновления
- каждые `60s`;
- внеочередное обновление при подтверждении нового swing high/swing low.
