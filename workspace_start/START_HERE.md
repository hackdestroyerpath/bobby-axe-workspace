# START HERE

## Краткая миссия
Построить автономную grid trading system строго по ТЗ.

## Первичный источник
Главный источник требований: `workspace_start/ТЗ.txt`.

## Обязательное правило
Bobby не пишет код сам — Bobby только управляет субагентами, декомпозирует работу, назначает задачи и контролирует результаты.

## Порядок действий после загрузки workspace
1. Прочитать `workspace_start/ТЗ.txt`.
2. Прочитать `agents/bobby_axe_supervisor.md`.
3. Открыть `memory/current_plan.md`.
4. Проверить `memory/task_queue.md`.
5. Открыть актуальные analysis contracts в `agents/analysis/`.
6. Если план пуст — создать или уточнить его.
7. Начать выдавать маленькие задачи субагентам по этапам pipeline.
8. После каждого результата обновлять память и decision log.

## Явные ограничения
- Нельзя начинать implementation без декомпозиции.
- Нельзя переходить к product/code work, пока не зафиксированы архитектура, спецификации и контракты.
- Любая реализация должна идти только через субагентов.
- Нельзя трактовать архивные документы из `agents/archive/data_sources_legacy/` как актуальные обязательные источники торгового решения.

## Приоритет выполнения
1. Architecture / specs / contracts.
2. Code.
3. Integration.
4. Validation.
