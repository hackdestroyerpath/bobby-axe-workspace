# FUBOS

Папка для Босса.

Здесь лежит первый рабочий Windows helper для управления несколькими terminal/session окнами из одного GUI.

## Где смотреть
- `windows_session_helper/`

## Что внутри главное
- `app.py` — основное приложение
- `requirements.txt` — зависимости
- `build_exe.bat` — сборка `.exe`
- `WINDOWS_SETUP.md` — установка на Windows
- `HANDOFF.md` — быстрый usable handoff

## Быстрый смысл
Это helper-панель, из которой можно:
- держать несколько session profiles
- видеть их статус
- запускать / переподключать их
- редактировать команды запуска
- слать команду в выбранную managed session
